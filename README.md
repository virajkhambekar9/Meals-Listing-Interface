# 🍽 Meals Listing App

## 🚀 Features

- Fetches meal data from Meals API
- Responsive grid layout
- Clean meal cards UI
- Displays:
  - Meal Image
  - Meal Name
  - Category
  - Area
  - Tags

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Meals API

## 📡 API Used

https://api.freeapi.app/api/v1/public/meals
